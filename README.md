# IRUN: Indulgent Reiz User Notation

A user-friendly query language, that transpiles
to ReizQL.


Write python-like DSL, get ReizQL version for free!
```py
def ...():
    for _ in range():
        ***
```

```py
FunctionDef(
    body=[
        For(
            target=Name(id="_"),
            iter=Call(
                func=Name(id="range")
            ),
        )
    ]
)
```
